「見せかけの回帰」

以下の2つのランダムウォークを回帰する．

\[ y_t=y_{t-1}+\epsilon_t =y_{t-2}+\epsilon_{t-1}+\epsilon_t=\cdots = \sum_{i=0}^t\epsilon_i \\ x_t=x_{t-1}+\epsilon'_t =x_{t-2}+\epsilon'_{t-1}+\epsilon'_t=\cdots = \sum_{i=0}^t\epsilon'_i \]

set.seed(123)
y_rw <- cumsum(rnorm(n_sample))
x_rw <- cumsum(rnorm(n_sample))
model_ols_rw <- lm(y_rw~x_rw)

Durbin-Watson(DW)検定

残差に自己相関があると，通常の最小自乗法は，問題が起きる(p125). これはのちにちゃんと説明する（多分）．

残差に自己相関があるかどうかをを検定する．

統計量は

\[ DW = \frac{\sum_{t=2}^T(\hat{u}_t-\hat{u}_{t-1} )^2}{\sum_{t=1}^T\hat{u}_t^2},\\ \mbox{where. } \hat{u}_t = y_t-\hat{\beta}_0+\hat{\beta}_1x_t \]

簡単のため，xは１次元っぽく書いているが，r次元でもいい．DW統計量は帰無仮説（自己相関）0のもとでおよそ２．

resid_ols <- model_ols_rw$residuals
dw <- sum(diff(resid_ols)^2) / sum(resid_ols^2)
dw
## [1] 0.05391462

大分2から離れているので自己相関はある．ちゃんと検定すると，

lmtest::dwtest(model_ols_wn)
## 
##  Durbin-Watson test
## 
## data:  model_ols_wn
## DW = 1.9363, p-value = 0.262
## alternative hypothesis: true autocorrelation is greater than 0
lmtest::dwtest(model_ols_rw)
## 
##  Durbin-Watson test

よくみると片側検定になっている．両側でやる．

200回試みる．

n_sim <- 200
n_sample <- 400
p_wn <- numeric(n_sim)
p_rw <- numeric(n_sim)
set.seed(1)
for(i in 1:n_sim){
  y_wn <- rnorm(n_sample)
  x_wn <- rnorm(n_sample)
  model_wn <- lm(y_wn~x_wn)
  p_wn[i] <- summary(model_wn)$coefficients["x_wn","Pr(>|t|)"]
  
  y_rw <- cumsum(rnorm(n_sample))
  x_rw <- cumsum(rnorm(n_sample))
  model_rw <- lm(y_rw~x_rw)
  p_rw[i] <- summary(model_rw)$coefficients["x_rw","Pr(>|t|)"]
}
sum(p_wn<0.05)/n_sim
## [1] 0.055
sum(p_rw<0.05)/n_sim
## [1] 0.85

  • 差分をとり自己相関をなくす．（実は共和分などの問題が起きうる）
